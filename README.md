MZANSI-REACT-APP 
Smart Shopping Made Local
MZANSI-REACT-APP is a mobile application built with React Native that empowers South African consumers to compare prices from their favorite stores all in one place. 

Features
Multi-store price comparison View prices for the same product across different retailers.

Favorites & Watchlist Save preferred stores and products to track price changes.
Real-time updates Stay informed with live price syncing and availability.

Clean, intuitive UI Built with React Native for smooth performance on Android and iOS.

Running the App
Make sure your in git bash
npx expo start
